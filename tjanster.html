Tjänster

Vi jobbar för att alla som använder våra tjänster och appar ska få ett så bra intryk som möjligt. Nu ska vi presentera vad vi presentera vad vi faktisk gör.

Exakthet

Vi erbjuder ett mycket exakt mät instrument som mäter längden på spjutet. Vi erbjuder även exaktdata på olika parametrar som kan förendra spjutet och ger fedback på kast efter med endast ett kanapptryk.

Underlättar utlärning

Vi erbjuder ett mät instrument som förenkla träningen för både aktiva ungdomar och deras tränare. Genom att erbjuda omedelbar återkoppling på hastighet, vinkel och längd med endast ett knapptryck, strävar vi efter att förbättra träningsprocessen.

Samhörighet

Vi på indoor distance erbjuder ett vertyg som alla kan ta del av. Tillsammans förenklar vi tränarnas och de aktivas utveckling. Föra all som provs till nybörjare.
